Professor Dumbledore has just been informed that Lord Voldemort has gathered a huge army of Wizards and Witches and is marching towards Hogwarts to attack from the North. To prepare for the attack, Dumbledore ordered all the Wizards and Witches at Hogwarts to spread across the Northern border of the school.

The border can be imagined as a straight line of length N units. At each unit a wizard is stationed.

Voldemort has a special way of attack. He sends a few people from his army every hour to attack, and the war will last for Q hours. On each hour he attacks a part of border between L and R units (both inclusive) with C wizards of class W.

Dumbledore knows about this attacking strategy. He Knows that in order to protect Hogwarts from getting destroyed he needs to have atleast C wizard of class W stationed in the part of the border that is being attacked. (i.e. between L and R)

The border is huge and there is no time to change the positions of wizards. You are the war tactician at Hogwarts and you have been given an Array A with array element A[i] denoting the class of wizard stationed at ith position and you have been asked to find whether the part that is being attacked will stand or not in a 'YES' or 'NO' format.

 

Input

The first line of the input contains an integer T denoting the number of test cases. The description of T test cases follows.

First line of each test case contains two space seperated integers N and Q denoting length of the border, and time period of war (in hours) respectively.

Next line of each test case contains N space seperated integers that form an array A, where A[i] denotes the class of wizard at ith unit,

The next Q lines of each test case contains 4 space separated integers L, R, W and C denoting start of range, end of range, class of attacking wizards and count of attacking wizards repectively.

 

Output

For each Query Q output a single line of either 'YES' or 'NO', without quotes.

Print 'YES' - if the attacking wizards can be defeated

Print 'NO' - if the wizards cannot be defeated

 

Constraints

  • 1 ≤ T ≤ 5
  • 1 ≤ N, Q ≤ 105
  • 1 ≤ A[i], X ≤ 109
  • 1 ≤ L ≤ R ≤ 105
  • 1 ≤ C ≤ R-L+1

 

Subtasks


Subtask #1 (30 points):

  • 1 ≤ N, Q ≤ 103
  • 1 ≤ L ≤ R ≤ 103
  • 1 ≤ C ≤ R-L+1

Subtask #3 (70 points):

  • Original Constraints

Example

Input:
1
5 3
1 2 3 2 1
0 1 1 1
1 4 2 2
1 4 2 3

Output:
YES
YES
NO

 

Explanation

Query 1:

For the first query, L=0, R=1, W=1, C=1. Since there is 1 wizard of class "1" present in range (0-1), which is >= C, the answer is "YES"

Query 2:

L=1, R=4, W=2, C=2. Since there are 2 wizards of class "2" present in range (1-4), which is >= C, the answer is "YES"

Query 3:

L=1, R=4, W=2, C=3. Since there are only 2 wizards of class "2 present in range (1-4), which is <= C, the answer is "NO"
