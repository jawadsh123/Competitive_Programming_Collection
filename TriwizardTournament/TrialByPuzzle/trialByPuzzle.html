In the same alternate Universe where the Grand Feast is taking place, Cersei has held Jon Snow captive. She has accused Jon of rebelling against the Crown by being Dany's ally. Jon denied saying that he was doing what needed to be done to fight off the White Walkers.

But Cersei wouldn't have it. Tyrion proposes a Trial by Combat. But as Cersei knows that Jon is a great warrior and has every chance of winning it, she spins up a new form of trial - The Trial by Puzzle. Cersei will propose a puzzle, if Jon fails to solve it, he'll be given the death penalty. If he wins on the other hand, he won't be killed but would still remain a prisoner.

Jon just needs to buy some time until Dany arrives with her Dragons so he accepts the offer.

The puzzle is as follows:

You are given a binary string of some radom length (i.e. the string is made up of only 1's and 0's). You need to transform this string into a string of only zeroes by performing certain operations.

In each operation, you can take a certain prefix of the string and flip all it's values. So now all the 1's in that prefix become 0's and vice-versa.

A prefix of string S is any string that starts from index 0 and ends at j, where j < length of S

The quest is to find the minimum number of such operations that would be required to make all the characters in the string 0's.

But, as you might have guessed, Jon knows nothing, so he has chosen you as his champion. Help him solve the puzzle or die with him ;(

 

Input

The first line of the input contains an integer T denoting the number of test cases. The description of T test cases follows.

Each test case has only one line denoting the binary string which Cersei gives to Jon

 

Output

For each test case output an integer representing the minimum number of operations required to transform the string into all 0's

 

Constraints

  • 1 ≤ T ≤ 10
  • 1 ≤ Slength ≤ 105

Subtasks


Subtask #1 (30 points):

  • 1 ≤ T ≤ 10
  • 1 ≤ Slength ≤ 103

Subtask #2 (70 points):

  • 1 ≤ T ≤ 10
  • Original Constraints

 

Example

Input:
1
010100

Output:
4

 

Explanation

Example case 1.

  • Operation 1: Flip all the values in the prefix of length 2 and transform it into 100100.
  • Operation 2: Flip all the values in the prefix of length 4 and transform it into 011000.
  • Operation 3: Flip all the values in the prefix of length 3 and transform it into 100000.
  • Operation 4: Flip all the values in the prefix of length 1 and transform it into 000000.

There are many other combination of operations possible, but none of then would take less than 4 operations so the answer is 4.
