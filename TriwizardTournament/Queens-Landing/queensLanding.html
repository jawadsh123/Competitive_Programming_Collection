The Dragon Queen (Danaerys) is about to land in King's Landing for meeting with the High Lords. Danaerys, flashy as usual, has come riding on a Dragon and she plans to land on the Dragon's Pit.

The Dragon's Pit is a giant rectangle made out of R*C square cells. (i.e. length=R, breadth=C)

Queen Cersei, having anticipated this, has filled the pit with Lannister Army. Each cell in the pit has a certain number of army men standing on it. So each cell has a number assigned to it that denotes the number of people standing on it.

Tyrion has reached by ship, and now witnesses the situation. He knows that only Dragon's pit is big enough to land a Dragon in King's Landing. So now he needs to clear a rectangle in the pit for the Queen to land. Luckily he has got some Wildfire at his disposal. To clear a particular rectangular section (sub-matrix) of the pit, x units of wildfire needs to be used.

For a particular sub-matrix, x = XOR sum of all the cells present in that region. Tyrion is trying to find a satisfactory sub matrix to clear. You will assist him in this task.

The problem will go as follows:

  • Tyrion needs to run Q queries
  • Each query will have four inputs - (x1, y1), (x2, y2), which are diagonally opposite points of a sub-matrix.
  • Your task is to find the number of units of Wildfire required to clear the sub-matrix formed by above points. (i.e. to find the XOR Sum of all the cells present in that sub-matrix)

With your crazy math skills, and Tyrion's intellect, we hope to see the Queen land and make History.

Input

The first line gives three space separated integers R, C, Q, giving the number of rows, number of columns and the number of Queries

The next R lines each contain, space separated C integers giving the elements of the rows The next Q lines each contain space separated four integers giving the top left and bottom right coordinates of the sub matrices (rows are numbered 0 to R­-1 and columns are numbered 0 to C­-1, and the top left corner of the matrix has coordinates 0, 0.

Output

Q lines each giving XOR sum of all the elements in the corresponding Sub Matrix.

Constraints

  • 5 ≤ R ≤ 1000
  • 5 ≤ C ≤ 1000
  • 5 ≤ Q ≤ 1000000

The elements of the matrix are integers in the range 0 to 100

Subtasks

  • Subtask 1 (10 points): 1 ≤ Q ≤ 100
  • Subtask 2 (90 points): Original Contraints

Examples -

Input:

Output:
14

Explanation

The first sub matrix is 

30 77 92
66 77 24
59 60 17

The XOR sum of these elements is 14. (i.e. 30^77^92^66^77^24^59^60^17 = 14)
